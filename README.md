# React + TypeScript + Vite + Eagle Plugin

- vite
- ts
- react
- tailwindcss
- daisyui

## Start

`use template button` or

```sh
git clone https://github.com/meetqy/eagle-plugin-vite-react-ts.git
```

**Install and dev**

```sh
pnpm i
pnpm dev
```

or

```sh
npm i
npm run dev
```
